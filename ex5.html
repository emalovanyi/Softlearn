1. Теорія

розв'язок нелінійних рівнянь програмним чином
Відомо, що для розв'язку нелінійного рівняння f(x) = 0 необхідно: знайти такі значення невідомого x, за яких рівняння перетворюється на тотожність.
Програмним чином, корені неліінійного рівняння можна вираховувати різними методами. Найпопулярнішими з яких є метод ділення навпіл та метод Ньютона.

метод ділення навпіл
він хоч і є відносно повільний, проте завжди приводить до шуканого результату. Його суть полягає в поділі заданого інтеервалу навпіл, до тих пір доки не буде локалізований корінь. алгоритм даного методу можна описати наступним чином
1. Задаємо інтервал [a,b], на якому імовірно існує тільки один корінь рівняння, і точність Eps обчислення значення кореня.
2. Перевіряємо, чи виконується умова: f(a) * f(b) "<" 0
3. Якщо умову (2) виконано, визначаємо середину інтервалу [a,b], тобто точку c = (a + b) / 2, і перевіряємо умову: f(a) * f(c) < 0.
4. Якщо вона виконується, то праву межу інтервалу переносимо в середню точку – шляхом присвоєння (b = с). У протилежному випадку, в точку с переносимо ліву межу ( a = c ).
5. Поділ уточненого інтервалу навпіл і перенесення однієї із меж інтервалу продовжуємо доти, доки не виконається умова: | b - a | < Eps .

Метод Ньютона
Ідея даного методу полягає в наступному
1. Вводимо межі a, b проміжку локалізації кореня, значення точності Eps та кількості ітерацій Kmax.
2. Надаємо змінній х значення b.
3. Якщо f(x)*f2p(x)"<"0, то надаємо змінній х значення а, інакше – переходимо до кроку 5;
4. Якщо f(x)*f2p(x)>0 то,
5. Починаємо цикл по і від 1 до Кmax, у тілі якого:
* обчислюємо Dx = f(x) / fp(x);
* виконуємо ітерацію Ньютона: x = x – Dx;
* якщо модуль значення Dx є менший за Eps це означає що корінь локалізовано

1. Завдання
Реалізуйте пошук коріня нелінійного рівняння, за допомогою методу ділення навпіл
1) для перевірки візьміть функцію f(x)= x * x - 4, і задайте точність 0,00001
2) межі пошуку задайте самостійно.
зауваження даний метод гарантує пошук тільки 1 кореня в заданому інтервалі

double f(double x, char k1) // ліві частини наших нелінійних рівнянь
                    {
                        return x * x - 4;
                    }
                    
                    double fp(double x, double d, char k1) // перша похідна функції f
                    {
                        return (f(x + d, k1) - f(x, k1)) / d;
                    }
                    double f2p(double x, double d, char k1) // друга похідна функції f
                    {
                        return (f(x + d, k1) + f(x - d, k1) - 2 * f(x, k1)) / (d * d);
                    }
                    double MDP(double a, double b, double Eps, char k1, int& L)
                    {
                        double c, Fc;
                        while (b - a > Eps)
                        {
                            c = a + 0.5 * (b - a);
                            L = L + 1; // лічильник кількості поділів інтервалу [a, b]
                            Fc = f(c, k1);
                            if (Math::Abs(Fc) < Eps) // перевірка, чи точка с не є поблизу кореня x*
                                return c; // завершення роботи функції MDP
                            if (f(a, k1) * Fc > 0)
                                a = c;
                            else b = c;
                        }
                        return c; // завершення роботи функції MDP
                    }

2. Завдання
реалізуйте метод Ньютона для пошуку кореня нелінійного рівняння.
1. Для перевірки візьміть ті ж числа що і в 1 завданні.

double f(double x, char k1) // ліві частини наших нелінійних рівнянь
                        {
                            return x * x - 4;
                        }
                        
                        double fp(double x, double d, char k1) // перша похідна функції f
                        {
                            return (f(x + d, k1) - f(x, k1)) / d;
                        }
                        double f2p(double x, double d, char k1) // друга похідна функції f
                        {
                            return (f(x + d, k1) + f(x - d, k1) - 2 * f(x, k1)) / (d * d);
                        }
                        double MN(double a, double b, double Eps, char k1, int Kmax, int& L)
                        {
                            double x, Dx, D; int i;
                            Dx = 0.0; D = Eps / 100.0;
                            x = b;
                            if (f(x, k1) * f2p(x, D, k1) < 0)
                                x = a;
                            if (f(x, k1) * f2p(x, D, k1) < 0)
                                MessageBox::Show("Для цього рівняння збіжність ітерацій не гарантована");
                            for (i = 1; i <= Kmax; i++)
                            {
                                Dx = f(x, k1) / fp(x, D, k1);
                                x = x - Dx;
                                if (Math::Abs(Dx) < Eps)
                                {
                                    L = i; // занесення у лічильник L результуючої кількості ітерацій
                                    return x; // завершення роботи функції MN
                                }
                            }
                            MessageBox::Show("За задану кількість ітерацій кореня не знайдено");
                            return -1000.0;
                        }
