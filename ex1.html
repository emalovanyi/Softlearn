1. Теорія
Сортування масиву це процес розподілу всіх елементів масиву в певному порядку і є однією з найбільш поширених операцій обробки даних.

Існує безліч способів сортування масивів. Одними з найпоширеніших є сортування бульбашкою, вставкою і вибором. Сортування масивів методом вибору, мабуть, найпростіше для розуміння, хоч і одне з найповільніших.
далі розглядатимуться алгоритми кожного з них.

Сортування вибором
Для сортування масиву шляхом вибору від найменшого до найбільшого елемента виконуються такі кроки:
1. Починаючи з елемента під індексом 0, шукаємо у масиві найменше значення.
2. Знайдене значення міняємо місцями із нульовим елементом.
3. Повторюємо кроки №1 і №2 для наступного індексу в масиві (відсортований елемент більше не чіпаємо).

Сортування бульбашкою
Суть полягає у порівнянні пари значень, які знаходяться поруч, і якщо задоволені задані критерії, значення з цієї пари змінюються місцями. І таким чином елементи "скачуть бульбашкою" до кінця масиву.
1. Порівнюється елемент масиву під індексом 0 з елементом масиву під індексом 1. Якщо елемент під індексом 0 більший за елемент під індексом 1, то значення змінюються місцями.
2. Потім ми переміщуємося до наступної пари значень: елемент під індексом 1 і елемент під індексом 2 і так до тих пір, поки не досягнемо кінця масиву.
3. Повторюємо крок №1 і крок №2 до тих пір, поки весь масив не буде відсортовано.

Сортування вставкою
В сортуванні методом вставки (англ. insertion sort) масив даних розділяється на дві частини: ліву – відсортовану і праву – невідсортовану. Перший елемент невідсортованої частини вставляється у відсортовану частину таким чином, щоб вона залишалася відсортованою. У результаті відсортована частина збільшується на один елемент, а невідсортована – зменшується.

1. Завдання
напишіть код програми за алгоритмом з підзаголовка «Сортування масивів методом вибору» так, щоб сортування виконувалося
а) в порядку зменшення (від найбільшого числа до найменшого)
б) в порядку збільшення (від найменшого числа до найбільшого)

#include <iostream> 
                        #include <algorithm> // для std::swap. У C++11 використовуйте заголовок <utility>
                         
                        int main()
                        {
                            const int length = 5;
                            int array[length] = { 30, 50, 20, 10, 40 };
                         
                            // Перебираємо кожен елемент масиву (крім останнього, він вже буде відсортований до того часу, коли ми до нього дійдемо)
                            for (int startIndex = 0; startIndex < length - 1; ++startIndex)
                            {
                                // У змінній smallestIndex зберігається індекс найменшого значення, яке ми знайшли в поточній ітерації.
                                // Починаємо з того, що найменший елемент в цій ітерації - це перший елемент (індекс 0)
                                int smallestIndex = startIndex;
                         
                                // Потім шукаємо менший елемент в іншій частині масиву
                                for (int currentIndex = startIndex + 1; currentIndex < length; ++currentIndex)
                                {
                                    // Якщо ми знайшли елемент, який менше нашого найменшого елементу,
                                    if (array[currentIndex] < array[smallestIndex])
                                        // то запам'ятовуємо його
                                        smallestIndex = currentIndex;
                                }
                         
                                // smallestIndex тепер найменший елемент.
                                // Міняємо місцями наше початкове найменше число з тим, яке ми виявили
                                std::swap(array[startIndex], array[smallestIndex]);
                            }
                         
                            // Тепер, коли весь масив відсортований - виводимо його на екран
                            for (int index = 0; index < length; ++index)
                                std::cout << array[index] << ' ';
                         
                            return 0;
                        }

2. Завдання
Напишіть програму, яка відсортує наступний масив бульбашковим сортуванням відповідно до вищевказаного алгоритму:
а) в порядку зменшення (від найбільшого числа до найменшого)
б) в порядку збільшення (від найменшого числа до найбільшого)

#include <iostream> 
                        #include <algorithm> // для std::swap. У C++11 використовуйте заголовок <utility>
                         
                        int main()
                        {
                            const int length(9);
                            int array[length] = { 7, 5, 6, 4, 9, 8, 2, 1, 3 };
                         
                            for (int iteration = 0; iteration < length-1; ++iteration)
                            {
                                // Перебираємо кожен елемент масиву до останнього елементу (не включно).
                                // Останній елемент не має пари для порівняння
                                for (int currentIndex = 0; currentIndex < length - 1; ++currentIndex)
                                {
                                    // Якщо поточний елемент більше елементу після нього, то міняємо їх місцями
                                    if (array[currentIndex] > array[currentIndex+1])
                                        std::swap(array[currentIndex], array[currentIndex + 1]);
                                }
                            }
                         
                            // Виводимо відсортовний масив на екран
                            for (int index = 0; index < length; ++index)
                                std::cout << array[index] << ' ';
                         
                            return 0;
                        }

а) в порядку зменшення (від найбільшого числа до найменшого)
б) в порядку збільшення (від найменшого числа до найбільшого)

template<typename Y>
                        void Array<Y>::sortInsert(int min, int max)
                        {
                            for (int i = min; i < max; i++)
                            {
                                int k = i;
                                while (k > min && m_array[k - 1] > m_array[k])
                                {
                                    swap(m_array[k - 1], m_array[k]);
                                    --k;
                                }
                            }
                        }
