Типи операцій

Арифметичні операції

  • + : операція додавання повертає суму двох чисел.
  • - : операція віднімання повертає різницю двох чисел.
  • * : операція множення повертає добуток двох чисел.
  • / : операція ділення повертає частку двох чисел.
  • % : операція отримання остачі від цілочисельного ділення.
    • Також є дві унарні арифметичні операції, які проводяться над одним числом: ++ (інкремент) та - (декремент). Кожна з операцій має два різновиди: префіксний та постфіксний:
  • Префіксний інкремент:
    Збільшує значення змінної на одиницю та отриманий результат використовується як значення виразу ++x.
  • Постфіксний інкремент: Збільшує значення змінної на одиницю, але значенням виразу x++ буде те, що було до збільшення на одиницю.
  • Префіксний декремент:
    Зменшує значення змінної на одиницю та отриманий результат використовується як значення виразу ++x.
  • Постфіксний декремент: Зменшує значення змінної на одиницю, але значенням виразу x++ буде те, що було до зменшення на одиницю.

    • Арифметичні операції обчислюються зліва направо. Одні операції мають більший пріоритет, ніж інші і тому виконуються спочатку. Операції у порядку зменшення пріоритету:

    • ++ (інкремент), -- (декремент)
    • * (множення), / (ділення), % (остача від ділення)
    • + (додавання), - (віднімання)
      • Порядок виконання арифметичних операцій змінюється за допомогою ().

    Умовні операції

    Умовні операції повертають значення типу bool, тобто true - коли умова істинна і false - якщо хибна. Умовні операції поділяються на:

    • Операції відношення:
  • ==: операція "дорівнює". Повертає true, якщо обидва операнди рівні, і false, якщо вони не рівні:
  • >: операція "більше ніж". Повертає true, якщо перший операнд більший за другий, і false, якщо перший операнд менший за другий.
  • <: операція "менше ніж". Повертає true, якщо перший операнд менший за другий, і false, якщо перший операнд більший за другий.
  • >=: операція "більше або рівне ніж". Повертає true, якщо перший операнд більший або рівний за другий, і false, якщо перший операнд менший за другий.
  • <=: операція "менше або рівне ніж". Повертає true, якщо перший операнд більший або рівний за другий, і false, якщо перший операнд більший за другий.
  • !=: операція "не дорівнює". Повертає true, якщо перший операнд не дорівнює другому, і false, якщо обидва операнди рівні.

    • Логічні операції:
  • !(логічне заперечення): унарна операція, яка повертає true, якщо операнд дорівнює false. Якщо операнд дорівнює true, то операція повертає false.
  • &&(кон'юкція, логічне множення): повертає true, якщо обидва операнди не рівні false. Повертає false, якщо хоча б один операнд дорівнює false.
  • ||(диз'юнкція, логічне додавання): повертає true, якщо хоча б один операнд дорівнює true. Повертає false, якщо обидва операнди рівні false.

    • Побітові операції

    Побітові операції виконуються над окремими розрядами чи бітами чисел. Дані операції здійснюються лише над цілими числами.

    • Операції зсуву:
  • << (логічне заперечення): унарна операція, яка повертає true, якщо операнд дорівнює false. Якщо операнд дорівнює true, то операція повертає false.
  • >> (кон'юкція, логічне множення): повертає true, якщо обидва операнди не рівні false. Повертає false, якщо хоча б один операнд дорівнює false.
    • Побітові операції:
  • & : порозрядна кон'юнкція. Повертає 1, якщо обидва з відповідних розрядів обох чисел дорівнюють 1.
  • | : порозрядна диз'юнкція. Повертає 1, якщо хоча б один із відповідних розрядів обох чисел дорівнює 1.
  • ^ : порозрядне виключає АБО. Повертає 1, якщо тільки один із відповідних розрядів обох чисел дорівнює 1.
  • ~ : порозрядне заперечення чи інверсія. Інвертує всі розряди операнда. Якщо розряд дорівнює 1, він стає дорівнює 0, і якщо він дорівнює 0, він отримує значення 1.

    • Операції присвоювання

      Операції приcвоювання дозволяють надати змінній деякого значення. Ці операції виконуються над двома операндами, причому лівий операнд може представляти тільки змінну. Базова операція приcвоювання = дозволяє надати значення правого операнда лівому операнду
    • +=: приcвоювання після додавання. Надає лівому операнду суму лівого та правого операндів: A += B еквівалентно A = A + B
    • -=: приcвоювання після віднімання. Надає лівому операнду різницю лівого і правого операндів: A -= B еквівалентно A = A - B
    • *=: приcвоювання після множення. Надає лівому операнду добуток лівого та правого операндів: A *= B еквівалентно A = A * B
    • /=: приcвоювання після ділення. Надає лівому операнду частку лівого та правого операндів: A /= B еквівалентно A = A / B
    • %=: приcвоювання після ділення з остачею. Надає лівому операнду остачу від цілочисельного ділення лівого операнда на правий: A %= B еквівалентно A = A % B
    • <<=: приcвоювання після зсуву розрядів ліворуч. Надає лівому операнду результат зсуву його бітового подання вліво на певну кількість розрядів, що дорівнює значенню правого операнда: A <<= B еквівалентно A = A << B
    • >>=: приcвоювання після зсуву розрядів праворуч. Надає лівому операнду результат зсуву його бітового подання вправо на певну кількість розрядів, що дорівнює значенню правого операнда: A >>= B еквівалентно A = A >> B
    • &=: приcвоювання після порозрядної кон'юнкції. Надає лівому операнду результат порозрядної кон'юнкції його бітового подання з бітовим поданням правого операнда: A &= B еквівалентно A = A & B
    • |=: приcвоювання після порозрядної диз'юнкції. Надає лівому операнду результат порозрядної диз'юнкції його бітового подання з бітовим подання правого операнда: A |= B еквівалентно A = A | B
    • ^=: приcвоювання після операції, що виключає АБО. Надає лівому операнду результат операції виключає АБО його бітового подання з бітовим подання правого операнда: A ^= B еквівалентно A = A ^ B
