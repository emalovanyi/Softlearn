1. Теорія

Комбінаторика
Комбінаторика – це розділ математики, що досліджує об’єкти зі скінчених множин та операції над ними.

основні комбінаторні об’єкти
Розглянемо основні комбінаторні об’єкти – розміщення та сполучення. Нехай задано скінченну n-елементну непорожню множину A a a a = { 1 2 , ,..., n }. 1. Розміщенням з n елементів по k будемо називати k елементний впорядкований набір елементів з множини А.
2. Сполученням (комбінацією) з n елементів по k будемо називати k-елементний невпорядкований набір елементів з множини А.
3. Якщо в розміщенні/сполученні дозволяється наявність однакових елементів, то такі розміщення/сполучення називаються з повтореннями. В іншому випадку говорять про розміщення/сполучення без повторень.

Генерування перестановок
Алгоритм генерування перестановки наступний
1. потрібно знайти в перестановці першу справа пару сусідніх елементів, у якій елемент, що ліворуч, менший від елементу, що праворуч.
2. Переставити місцями елемент а та найменший з елементів серед наступних елементів, який водночас є більшим, ніж a .
3. Записати у висхідному порядку (посортувати за зростанням) всі наступні псля а елементи .

Генерування сполучень
алгоритм генерування сполучень наступний
1. Знайти в сполучені перший справа елемент a такий, що a ≠ n − k + i, де n – кількість елементів вихідної множини A, k – кількість елементів сполучення, i – номер (позиція) елемента. Величина n − k + i дорівнює максимально можливому значенню елементу a в і-тій позиції.
2. Збільшити знайдений елемент на одиницю.
3. Для кожного наступного елементу встановити його значення рівне значенню попереднього елемента, збільшеного на одиницю.

1. Завдання
Факторіал цілого числа N визначається як добуток всіх послідовних чисел від 1 до N (0! = 1).
1) Напишіть рекурсивну функцію factorial(), яка повертає факторіал вводу
2) Протестуйте її за допомогою перших 8 чисел. 

#include <iostream>
 
                        int factorial(int n)
                        {
                            if (n < 1)
                                return 1;
                            else
                                return factorial(n - 1) * n;
                        }
                         
                        int main()
                        {
                            for (int count = 0; count < 8; ++count)
                                std::cout << factorial(count) << '\n';
                        }

2. Завдання
реалізуйте генерування перестановок, за наведеним вище алгоритмом.
1. Для перевірки візьміть множину 1, 2, 3, 4, 5, 6.
2. на останній ітерації вона повинна набути зворотнього вигляду. 

void Comb::genPerm()
                        {
                            int index;
                            for (int iii = m_n - 2; iii >= 0; iii--)
                            {
                                if (array[iii] < array[iii + 1])
                                {
                                    index = iii;
                                    break;
                                }
                            }
                            sort(array + index + 1, array + m_n);
                            int need = index + 1;
                            while (need < m_n)
                            {
                                if (array[index] < array[need])
                                {
                                    swap(array[index], array[need]);
                                    break;
                                }
                                need++;
                            }
                        }

3. Завдання
згенеруйте сполучення, за допомогою алгоритму який детально описаний вище.
1) для перевірки візьміть множину {1, 2, 3, 4, 5, 6} n=6, k=4
2) в результаті, на останній ітерації має вийти множина {3, 4, 5, 6}

void Comb::genComb()
                        {
                            for (int i = m_k - 1; i >= 0; i--)
                            {
                                if (array[i] < m_n - m_k + i + 1)
                                {
                                    ++array[i];
                                    for (int j = i + 1; j < m_k; j++)
                                        array[j] = array[j - 1] + 1;
                                    return;
                                }
                            }
                        }
