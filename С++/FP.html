Перша програма в
  • Спочатку вам слід обрати редактор коду,як-от: Visual Studio,Intellij Idea,Sublime Text,NotePad++...
  • Далі створюємо консольний проект C++ (файл має розширення .cpp).
    Для виведення рядка на консоль необхідно підключити потрібний функціонал. Для цього на початку файлу йде рядок:
    • #include <iostream>


  • Далі йде визначення функції main. Функція main повинна бути присутня в будь-якій програмі на С++, з неї власне і починається виконання програми.

    Функція main складається з чотирьох елементів:

  • Тип значення, що повертається(int).
  • Назва функції(main).
  • Список параметрів. (int main(//параметри функції))
  • Тіло функції. ({//набір операторів})

    У тілі функції відбувається виведення рядка на консоль. Для звернення до консолі використається стандартний потік виведення std::cout(std- "Простір імен"). За допомогою оператора << в цей потік (в даному випадку фактично на консоль) передається рядок символів, який треба вивести на консоль, тобто Hello world!

    Наприкінці здійснюємо вихід із функції за допомогою оператора return. Так як функція повинна повертати ціле число, то після return вказується число 0. Нуль використовується як індикатор успішного завершення програми.

    Після кожної інструкції у мові C++ ставляться крапка з комою.

    Повний текст програми:

    • #include <iostream>               // підключаємо бібліотеку iostream
 
        int main()                          // оголошуємо функцію main
        {                                   // початок функції 
            std::cout << "Hello World!";   // виводимо повідомлення на консоль
            return 0;                       // вихід з функції
        }








    Також можна вивести повідомлення способом введення змінної: 
    #include <iostream>   // підключаємо бібліотеку iostream
        #include <string>       // підключаємо бібліотеку типу string
        int main()               // оголошуємо функцію main
        {      
          std::string x="Hello,It's C++";  // оголошуємо змінну типу string та ініціалізуємо її(присвоюємо для неї значення)
            std::cout <<x;          // виводимо змінну на консоль
            return 0;                    // вихід з функції
        }









  • Для кращої навігації в коді використовують коментарі.Все, що написано після подвійного слешу //(/**/-для 2 і більше рядків) є коментарем. Коментарі ніяк не впливають на роботу програми(компілятор ігнорує їх), і дозволяють зрозуміти, що відбувається в певній стрічці коду.
